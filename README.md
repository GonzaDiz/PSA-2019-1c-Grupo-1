# PSA-2019-1c-Tribu-1

## Dependencias:

### NPM

Ubuntu

```
$ sudo apt install npm
```

MacOS

```
$ brew install npm
```


### Maven

Ubuntu

```
$ sudo apt install maven
```

MacOS

```
$ brew install maven
```


## Correr el proyecto React en localhost:8080 :

```
$ npm install
$ npm run dev
```

## Levantar todo el proyecto en localhost:5000:

```
$ mvn clean package
```
